# 高考志愿填报助手 (Gaokao Volunteer Assistant)

一个面向 AI Agent 的 Skill 定义，专注于**高考志愿填报**场景。帮助 AI 根据考生的分数、位次、省份、科类、兴趣偏好等信息，提供结构化的院校与专业填报建议，并解释"冲、稳、保"的梯度逻辑。

## 用途

当用户咨询以下问题时自动触发：

- 高考志愿填报建议
- 选大学、选专业指导
- 冲稳保填报策略分析
- 录取概率与梯度评估
- 根据分数/位次推荐院校

## 文件结构

```
gaokao-volunteer-assistant/
├── SKILL.md                          # Skill 主定义文件（YAML frontmatter + 工作流）
├── README.md                         # 本文件
└── references/
    └── 高考志愿填报通用知识.md          # 高考志愿填报参考资料
```

### SKILL.md

核心文件，包含：

- **YAML frontmatter**：name 和 description，用于 AI Agent 自动匹配触发
- **五步工作流**：
  1. 信息采集（省份、科类、总分、位次、选科、意向等）
  2. 梯度策略分析（冲、稳、保）
  3. 输出推荐表格（6 列标准格式）
  4. 风险提示（免责声明 + 注意事项）
  5. 边界声明（数据来源、能力边界）
- **使用示例与注意事项**

### references/

参考知识文档，整理了高考志愿填报的通用知识要点：

- 平行志愿规则
- 冲稳保策略详解
- 专业级差说明
- 服从调剂利弊分析
- 提前批与普通批区别
- 位次 vs 分数
- 征集志愿说明
- 新高考改革要点
- 常见误区提醒

## 安装方法

### 方式一：直接复制到 Trae IDE 的 skills 目录

```bash
# 将本项目克隆或下载到本地
git clone https://github.com/你的用户名/gaokao-volunteer-assistant.git

# 复制到 Trae IDE 的 skills 目录
cp -r gaokao-volunteer-assistant /path/to/.trae/skills/
```

### 方式二：Git 子模块（推荐多人协作）

```bash
cd /path/to/你的项目/.trae/skills/
git submodule add https://github.com/你的用户名/gaokao-volunteer-assistant.git
```

## 使用示例

**用户输入**：

> 我是广东物理类考生，598 分，位次 28000，选科物化生，想去广深，学计算机或电子信息，院校优先。

**Skill 响应摘要**：

1. 确认信息 → 确认冲稳保比例（院校优先可适当增加"冲"的比例）
2. 给出推荐表格（示例）：
   - **冲**：华南理工大学（部分专业）、暨南大学
   - **稳**：深圳大学、华南师范大学
   - **保**：广州大学、广东工业大学
3. 附带详细的风险提示和免责声明

## 核心特色

- ✅ 结构化信息采集，不漏关键参数
- ✅ 冲稳保梯度策略 + 比例建议
- ✅ 6 列标准输出表格（院校名称、专业、城市、参考分数/位次、梯度、推荐理由）
- ✅ 完整风险提示（专业级差、调剂、体检受限等）
- ✅ 边界清晰，不编造数据，如实说明数据来源
- ✅ 全部内容简体中文

## 数据声明

本 Skill **不包含** 任何具体院校的实时录取分数线数据。所有推荐建议均基于：

- 用户提供的分数和位次信息
- 通用填报策略规则
- 用户指定的偏好条件

具体分数线数据请以各省教育考试院和高校官方发布的招生计划为准。

## 注意事项

- 本 Skill 提供的建议仅供参考，不构成填报决策的唯一依据
- 高考政策每年可能调整，请以当年官方文件为准
- 建议结合《志愿填报指南》、高校招生章程等多方信息综合决策
- 推荐结果受限于 AI 的知识边界，可能不包含最新招生变动
