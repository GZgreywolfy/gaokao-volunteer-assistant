# 平行志愿规则

> **免责声明**：本文档为通用知识整理，具体政策以各省教育考试院和各高校当年发布的官方招生文件为准。高考政策每年可能调整，请务必查阅最新官方信息。

---

## 什么是平行志愿

平行志愿是指考生在填报志愿时，可在同一批次同时填报若干个平行院校志愿。投档时按照"**分数优先、遵循志愿、一轮投档**"的原则进行。

## 核心原则

| 原则 | 说明 |
|------|------|
| **分数优先** | 省教育考试院按考生总分从高到低排序，分数高的考生优先获得投档机会 |
| **遵循志愿** | 对单个考生而言，按照其所填报的志愿顺序依次检索，一旦检索到符合条件的志愿即投档 |
| **一轮投档** | 在同一批次中，考生档案只能投出一次。若投档后被退档，则无法再投同一批次的其他志愿 |

## 平行志愿的关键注意事项

- **志愿有顺序**：虽然是"平行"，但检索时仍有先后顺序，理想院校应放在前面
- **退档风险依然存在**：投档后如果因为身体条件、单科成绩不达标、不服从调剂等原因被退档，只能参加征集志愿或下一批次
- **不存在"投档后再投"**：一轮投档意味着档案只有一次投出机会

## 平行志愿 vs 顺序志愿

| 对比项 | 平行志愿 | 顺序志愿（传统梯度志愿） |
|--------|---------|----------------------|
| 投档原则 | 分数优先、遵循志愿 | 志愿优先、从高分到低分 |
| 考生风险 | 退档后无法补投 | 第二志愿有效概率大幅降低 |
| 适用范围 | 绝大多数省份的本科批/专科批 | 部分省份的提前批或特定批次 |
| 志愿数量 | 较多（视省份不同从十几个到上百个不等） | 通常较少（1-3 个院校志愿） |

## 位次在平行志愿中的核心作用

平行志愿模式下，位次是决定投档顺序的最关键指标。同一省份同一科类的考生按位次排队，位次高的考生优先获得检索机会。因此：

- **位次精准度**：填报时以位次为主要参考，分数仅作为辅助
- **院校定位**：参考目标院校近 3 年的录取位次区间进行匹配
- **大小年判断**：结合位次波动而非分数波动来判断院校冷热变化
