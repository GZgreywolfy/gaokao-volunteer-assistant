# 提前批与特殊类型招生

> **免责声明**：本文档为通用知识整理，内容仅供参考。具体政策以各省（自治区、直辖市）教育考试院和各高校当年发布的官方招生文件为准。高考政策每年可能调整，请务必查阅最新官方信息。

---

## 一、提前批概述

### 提前批的性质

提前批是在普通批次之前录取的批次，包含军事、公安、师范、医学等特殊类型招生。

### 提前批包含的招生类型

| 类型 | 说明 | 特殊要求 |
|------|------|---------|
| 军事院校 | 国防科大等军队院校 | 需通过政审、军检、面试，毕业后服役 |
| 公安政法类 | 中国人民公安大学等 | 需通过政审、体测、面试 |
| 公费师范生 | 部属/省属师范院校 | 免学费、包分配，有 6 年服务年限要求 |
| 免费医学生 | 定向培养基层医疗人才 | 服务基层医疗机构 |
| 航海类 | 大连海事大学等航海专业 | 对身体条件有特殊要求 |
| 小语种 | 部分外语院校提前招生 | 部分需要口语考试 |
| 综合评价录取 | 结合高考+校测+学考成绩 | 需提前报名并参加校测 |
| 强基计划 | 基础学科拔尖人才培养 | 高考后校测，本硕博衔接培养 |
| 定向招生 | 面向特定地区或单位的定向就业 | 毕业后须到定向单位/地区工作 |

### 提前批特点

- **录取在前**：提前批录取结束后才开始普通批
- **不影响后续批次**：提前批未被录取，可正常参加普通批
- **一旦录取不可放弃**：被提前批录取后，不再参加后续批次
- **部分院校有特殊要求**：如军检、政审、面试等

### 提前批的选择建议

- 对军警、师范、医学等有明确意向的考生可重点考虑提前批
- 提前批"不报白不报"是误区——如果被录取将失去普通批机会
- 建议结合自身职业规划慎重决定是否填报提前批
- 提前批的热门专业分数可能不低于甚至高于普通批同层次院校

## 二、强基计划

### 什么是强基计划

强基计划（基础学科招生改革试点）于 2020 年起实施，聚焦数学、物理、化学、生物、文史哲等基础学科，选拔培养有志于服务国家重大战略需求的拔尖人才。

### 强基计划要点

| 项目 | 说明 |
|------|------|
| **招生院校** | 39 所"双一流"高校 |
| **招生专业** | 数学、物理、化学、生物、文史哲等基础学科，部分高校新增交叉学科 |
| **报名时间** | 通常为 4 月（高考前） |
| **录取方式** | 高考成绩（占 85%）+ 校测成绩（占 15%）综合录取 |
| **培养模式** | 小班化、导师制、本硕博衔接培养 |
| **限制条件** | 录取后原则上不能转到非强基专业；退出强基需补缴相关费用 |

### 强基计划填报建议

- 适合对基础学科有浓厚兴趣、有志于科研的考生
- 强基计划在提前批之前录取（部分省份为单独批次）
- 报名需在高考前完成，错过无法补报
- 录取后转专业受限，需慎重选择

## 三、综合评价录取

### 什么是综合评价

综合评价录取是指高校结合考生高考成绩、高校校测成绩、高中学业水平考试成绩等多方面因素进行综合评价，择优录取。

### 综合评价要点

| 项目 | 说明 |
|------|------|
| **招生院校** | 部分双一流高校和省属重点高校 |
| **录取权重** | 常见比例：高考 60% + 校测 30% + 学考 10%（各校不同） |
| **报名时间** | 多数在 4-5 月（高考前） |
| **校测形式** | 面试为主，部分院校有笔试 |

### 综合评价填报建议

- 适合综合素质突出、高考成绩在院校边缘的考生
- 需提前关注目标院校的招生简章，按时报名
- 校测（面试/笔试）成绩对最终录取影响较大
- 与强基计划可同时报考（录取顺序不同）

## 四、定向招生

### 什么是定向招生

定向招生是指为特定地区或单位定向培养人才，考生毕业后须到指定单位或地区工作的招生方式。

### 主要定向类型

| 类型 | 说明 | 服务要求 |
|------|------|---------|
| **定向就业** | 毕业后到指定单位工作 | 通常 5-8 年 |
| **定向地区** | 面向特定地区（如贫困地区）招生 | 毕业后回生源地工作 |
| **定向培养** | 如定向医学生、定向师范生 | 服务基层若干年 |

### 定向招生填报建议

- 定向招生通常在提前批录取
- 享受学费减免或补助，但需承担服务义务
- 录取分数通常低于普通批同类专业
- 适合愿意在定向地区/单位发展的考生
- 违约需承担相应责任（记入诚信档案、退还补助等）

## 五、各批次录取顺序（以常见安排为例）

```
强基计划录取
    ↓
提前批录取（军校、公安、公费师范、定向等）
    ↓
本科批录取（普通批）
    ↓
专科批录取
```

> **注意**：各省批次设置和录取顺序可能存在差异，请以当年省教育考试院公布的时间安排为准。
