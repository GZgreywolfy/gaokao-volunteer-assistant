# 高考志愿填报通用知识

> **免责声明**：本文档为通用知识整理，内容仅供参考。具体政策以各省（自治区、直辖市）教育考试院和各高校当年发布的官方招生文件为准。高考政策每年可能调整，请务必查阅最新官方信息。

---

## 一、录取批次

全国各批次设置因省份而异，一般包括：

- **提前批**：军事、公安、师范、医学等特殊类型招生，在普通批次之前录取
- **本科批**：普通本科院校招生，新高考改革省份已逐步合并本科一批/二批
- **专科（高职）批**：专科层次院校招生

> 具体批次设置以本省教育考试院公布的官方文件为准。

## 二、核心概念速查

### 平行志愿

同一批次可填报多个平行志愿，投档遵循"**分数优先、遵循志愿、一轮投档**"原则。详见 [平行志愿规则.md](平行志愿规则.md)。

### 顺序志愿

以"志愿优先、从高分到低分"为原则，第一志愿至关重要。常见于提前批、艺术类部分批次。

### 征集志愿

各批次正常录取结束后，未完成招生计划的院校公布缺额，面向符合条件的考生再次填报志愿。

### 投档线

省教育考试院按招生计划数和投档比例向院校投档的最低分数。

### 省控线

省教育考试院划定的各批次最低录取控制分数线。

### 一分一段表

省教育考试院公布的各分数段对应累计人数统计表，用于判断考生位次。

## 三、常见术语速查表

| 术语 | 含义 |
|------|------|
| 滑档 | 考生的分数未达到所填报所有院校/专业组的投档线，档案未被投出 |
| 退档 | 考生档案已被投至某院校/专业组，但因不服从调剂、身体条件不符等原因被退回 |
| 投档线 | 省教育考试院按招生计划数和投档比例向院校投档的最低分数 |
| 位次 | 考生在所在省份全体考生中的排名，较分数更具参考价值 |
| 省控线 | 省教育考试院划定的各批次最低录取控制分数线 |
| 一分一段表 | 省教育考试院公布的各分数段对应累计人数统计表，用于判断考生位次 |
| 专业调剂 | 考生未被所填报专业录取时，由院校在本批次在本省招生的同类别专业范围内调剂安排 |
| 征集志愿 | 各批次录取结束后，未完成招生计划的院校向社会公布缺额，供未被录取的考生补填 |
