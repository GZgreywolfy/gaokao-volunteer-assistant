# 高考志愿填报通用知识

> **免责声明**：本文档仅供高考志愿填报参考之用，所有内容均为通用知识梳理，不构成任何具体的填报建议。文中不包含任何高校具体的分数线、位次等实时数据。各年份政策细节请以考生所在省（自治区、直辖市）教育考试院官方公布为准。

---

## 一、高考志愿填报基本概念

### 1.1 录取批次
- 全国各批次设置因省份而异，一般包括：**提前批、本科批（本科一批/二批或合并为本科批）、专科（高职）批**等。
- 新高考改革省份逐步合并本科批次，具体以本省教育考试院公布的批次设置为准。
- 部分省份仍保留"本科提前批""本科批""专科提前批""专科批"等划分。

### 1.2 平行志愿
- **核心原则**：分数优先、遵循志愿、一轮投档。
- "分数优先"：对同一批次所有考生按分数从高到低排队，高分者优先检索。
- "遵循志愿"：对每个考生，按其所填报的志愿顺序依次检索，一旦出现符合投档条件的院校（或专业组），即投档。
- "一轮投档"：在同一批次中，考生档案只投一次。若投档后因故被退档（如不服从专业调剂、身体条件不符等），则不再补投本批次后续志愿，只能参加征集志愿或下一批次。

### 1.3 顺序志愿（梯度志愿）
- **核心原则**：志愿优先、从高分到低分。
- 第一志愿至关重要。若第一志愿未录取，后续志愿被录取的概率大幅降低。
- 常见于提前批、艺术类部分批次。

### 1.4 征集志愿
- 各批次正常投档录取结束后，部分院校未完成招生计划，由省教育考试院统一公布，面向符合条件的考生再次填报志愿。
- 征集志愿是未被录取考生的补录机会，但院校和专业选择范围相对有限。

## 二、投档与录取规则

### 2.1 投档比例
- 平行志愿批次投档比例一般控制在 **105%以内**（即招收100人，最多调阅105名考生的档案），部分省份已按 **1:1** 投档。
- 顺序志愿批次的投档比例由招生院校确定，通常在 **120%以内**。

### 2.2 专业录取规则
考生档案投档到高校后，高校按以下规则之一进行专业录取：

- **分数优先（分数清）**：对所有进档考生按分数高低排序，依次满足专业志愿。高分考生的专业志愿优先得到满足。
- **专业志愿优先（专业清）**：先检索所有考生的第一专业志愿，按分数高低录取；未满额的专业再依次检索第二、第三专业志愿。第一专业志愿的选择至关重要。
- **专业级差**：考生未被第一专业志愿录取时，将其总分减去一个固定差值（如3分、2分、1分），再参与第二专业志愿的竞争。不同院校设置的级差分值不同，需查看招生章程。

### 2.3 退档原因
- 不服从专业调剂，且所填报的专业志愿均无法满足。
- 身体条件不符合所报专业要求（如色盲、色弱、身高限制等）。
- 单科成绩未达到专业录取要求。
- 外语语种不符合招生章程规定。

## 三、院校专业组填报详解

### 3.1 什么是"院校专业组"

- **定义**：院校专业组是新高考改革省份（广东、湖北、湖南、福建、江苏等）采用的志愿填报单位。一所院校可设置一个或多个"院校专业组"。
- **组成**：每个专业组包含若干专业（类），且对该组内的所有专业提出统一的选考科目要求。同一院校内，选考科目要求不同的专业分属不同的专业组。
- **定位**：以"院校专业组"为志愿单位，替代了传统的"以院校为志愿单位"模式。一个"院校专业组"即一个独立的投档单元。

### 3.2 填报要点

- 考生以"院校专业组"为单位填报志愿，而非以整所院校为单位。
- 同一院校的不同专业组，在投档时对应不同的投档分数线，投档结果相互独立。
- **选科要求必须匹配**：考生的选考科目须完全满足专业组的选科要求，否则无法投档至该专业组。
- **调剂在专业组内进行**：若考生被投档至某专业组但未被所填报的专业录取，将在该专业组内未录满的专业中进行调剂，**不会跨组调剂**到同一院校的其他专业组。

### 3.3 策略建议

- **"冲-稳-保"按专业组规划**：建议对每个专业组单独进行"冲-稳-保"定位，而非以整所院校为单位笼统判断。
- **关注专业组内专业构成**：仔细查看专业组内包含的全部专业，确认是否有自己不能接受的专业。若组内有"绝对不想去"的专业，需慎重考虑是否填报该组，以免被调剂到不喜欢的专业。
- **充分利用志愿数量**：在实行院校专业组模式的省份，考生可填报的志愿数量较多（有的省份可达45个专业组志愿），建议合理拉开梯度，充分覆盖不同层次的院校和专业组。

## 四、强基计划与综合评价简介

### 4.1 强基计划

- **选拔对象**：有志于服务国家重大战略需求，在基础学科（数学、物理、化学、生物、文史哲等）有突出才能或浓厚兴趣的学生。
- **录取方式**：高考成绩（占总成绩比例不低于 **85%**）+ 高校校测（笔试/面试/体测）。高校校测由各招生院校自主组织。
- **招生院校**：原 **39所** "双一流"A类高校，具体以当年教育部公布的招生院校名单为准。
- **招生专业**：聚焦数学、物理、化学、生物、力学、基础医学、历史、哲学、古文字学等基础学科领域。部分院校近年来在优势工科专业也有试点。
- **时间节点（一般流程）**：
  - **3月底-4月**：高校发布强基计划招生简章。
  - **4月**：考生在阳光高考平台等官方渠道网上报名。
  - **6月**：参加全国统一高考。
  - **6月下旬（高考出分后）**：确认并参加高校组织的校测（笔试、面试、体育测试）。
  - **7月**：在提前批录取开始前完成强基计划录取。
- **培养特点**：单独编班、配备一流师资、实行导师制、优先保研直博、本硕博衔接培养。强基计划录取考生原则上不得转专业，或仅限在强基计划专业内流转。

### 4.2 综合评价

- **选拔对象**：综合素质全面发展、在学业水平、社会实践、学科特长等方面表现突出的学生。
- **录取方式**：采用"高考成绩 + 校测成绩 + 学业水平测试成绩"三部分综合折算，常见比例：
  - 高考成绩通常占 **60%**
  - 高校校测成绩占 **30%**
  - 学业水平考试（合格考）成绩占 **10%**
  - 各省及各校比例略有不同，具体以招生简章为准。
- **典型省份**：江苏、山东、上海、浙江、广东等。部分院校面向多省招生，部分仅面向本省。
- **招生院校**：包括部分"双一流"高校和省属重点高校。
- **时间节点（一般流程）**：
  - **4-5月**：高校发布综合评价招生简章并开放报名系统。
  - **5月底**：高校公布初审通过名单。
  - **6月**：高考结束后，参加高校组织的校测（面试为主，部分院校含笔试）。
  - **6月底-7月初**：确认综合评价志愿（部分省份在提前批或单独批次填报）。

### 4.3 两者区别

| 维度 | 强基计划 | 综合评价 |
|------|----------|----------|
| 侧重方向 | 基础学科拔尖人才 | 综合素质全面 |
| 高考成绩占比 | ≥85% | 约60% |
| 招生院校 | 原39所双一流A类高校 | 双一流及省属重点高校 |
| 培养模式 | 本硕博衔接培养 | 普通本科培养为主 |
| 专业范围 | 基础学科为主 | 院校优势专业较广 |

## 五、艺术体育类志愿规则要点

### 5.1 艺术类

- **双线要求**：艺术类考生须同时达到 **专业考试（省统考/校考）合格线** 和 **文化课录取控制线**，两条线均达标方可参与投档录取。
  - 省统考：各省教育考试院统一组织的艺术类专业考试。
  - 校考：经批准的独立设置艺术院校或参照院校自行组织的专业考试，一般要求省统考合格后方可参加。
- **综合分计算**：
  - 各省计算公式不同，常见的组合方式为：**文化成绩 × 比例 + 专业成绩 × 比例**。
  - 典型比例（文化:专业）有：**4:6、5:5、3:7** 等，部分省份还有文专折算后再加权的特殊公式。
  - **具体计算公式须查阅本省教育考试院公布的当年艺术类招生工作规定。**
- **志愿模式**：
  - 多数省份艺术类批次实行 **平行志愿**，按综合分排序投档。
  - 部分省份（或部分批次，如校考批次）仍实行 **顺序志愿（梯度志愿）**，第一志愿至关重要。
- **录取批次**：艺术类录取批次一般单列，在普通类批次之前进行。常见的批次设置有：艺术类提前批、艺术类本科批、艺术类专科批等。

### 5.2 体育类

- **双线要求**：体育类考生须同时达到 **体育专业统考成绩合格线** 和 **文化课成绩录取控制线**。
- **综合分计算**：综合分计算方式由各省制定，常见为文化成绩与体育专业成绩按一定比例合成。
- **录取批次**：体育类录取批次一般单列，与普通类分开投档录取。
- **志愿模式**：多数省份实行平行志愿。

### 5.3 注意事项

- **兼报规则**：艺术类/体育类考生一般也可兼报普通类志愿，具体兼报规则以各省教育考试院规定为准（如部分省份在兼报时有批次限制）。
- **单科成绩要求**：部分院校艺术类专业对单科成绩有要求，如外语成绩不低于一定分数。需仔细阅读各校招生章程。
- **校考确认**：通过了校考并获得合格证的考生，需特别注意各省规定的确认志愿时间节点，避免错过填报。
- **录取顺序**：艺术/体育类录取一般先于普通类。若已被艺术/体育类录取，则不再参与后续普通类批次的录取。

## 六、填报策略参考

### 6.1 "冲-稳-保"梯度策略
- **冲**：选择往年录取位次略高于考生位次的院校/专业组，大胆尝试，但不抱必然录取的希望。
- **稳**：选择往年录取位次与考生位次基本匹配的院校/专业组，录取概率较大。
- **保**：选择往年录取位次明显低于考生位次的院校/专业组，确保有学可上，避免滑档。

### 6.2 志愿填报注意事项
- 认真阅读目标院校的《招生章程》，了解专业录取规则、身体条件要求、单科成绩要求、学费标准等关键信息。
- 区分"分数优先""专业志愿优先""专业级差"等不同的专业分配规则，合理排列专业志愿顺序。
- 慎重决定是否勾选"服从专业调剂"。服从调剂可降低退档风险，但可能被录取到不喜欢的专业；不服从调剂则退档风险较高。
- 结合家庭情况、地域偏好、未来就业及深造规划等综合因素做出选择。

### 6.3 信息查询渠道
- **官方信息来源**：各省（自治区、直辖市）教育考试院官方网站及官方公众号、阳光高考平台。
- **高校信息**：目标院校的本科招生网、官方招生微信公众号。
- **重要文档**：《招生章程》《招生计划汇编》《历年录取分数统计》等，均以官方发布为准。

## 七、常见术语解释

| 术语 | 含义 |
|------|------|
| 滑档 | 考生的分数未达到所填报所有院校/专业组的投档线，档案未被投出 |
| 退档 | 考生档案已被投至某院校/专业组，但因不服从调剂、身体条件不符等原因被退回 |
| 投档线 | 省教育考试院按招生计划数和投档比例向院校投档的最低分数 |
| 位次 | 考生在所在省份全体考生中的排名，较分数更具参考价值 |
| 省控线 | 省教育考试院划定的各批次最低录取控制分数线 |
| 一分一段表 | 省教育考试院公布的各分数段对应累计人数统计表，用于判断考生位次 |
| 专业调剂 | 考生未被所填报专业录取时，由院校在本批次在本省招生的同类别专业范围内调剂安排 |
| 征集志愿 | 各批次录取结束后，未完成招生计划的院校向社会公布缺额，供未被录取的考生补填 |
