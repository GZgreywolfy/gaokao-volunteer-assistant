# 专业级差与调剂

> **免责声明**：本文档为通用知识整理，具体政策以各省教育考试院和各高校当年发布的官方招生文件为准。高考政策每年可能调整，请务必查阅最新官方信息。

---

## 一、专业级差

### 什么是专业级差

专业级差是指高校在录取考生专业时，若第一专业志愿未录取，则需将考生的投档分减去一个固定分数（级差分），再与其他考生竞争后续专业志愿。

### 举例说明

某高校设置的专业级差为 **3-1-1-0-0**（即第 1-2 专业志愿级差 3 分，第 2-3 专业志愿级差 1 分，以此类推）：

1. 考生 600 分，第一专业志愿未录取
2. 检索第二专业志愿时，按 597 分（600-3）参与竞争
3. 若第二专业志愿仍未录取，检索第三专业志愿时按 596 分（597-1）参与竞争
4. 以此类推

### 常见的级差设置类型

| 级差类型 | 模式示例 | 说明 |
|---------|---------|------|
| 递减型 | 3-1-1-0-0 | 最常见，级差逐渐减小 |
| 固定型 | 1-1-1-1-1 | 所有专业志愿间级差相同 |
| 混合型 | 5-3-1-0-0 | 前几个专业志愿级差较大 |
| 无级差 | 0-0-0-0-0 | 分数优先，不设级差（分数清） |

### 应对策略

- 优先填报最想去的专业
- 拉开专业志愿之间的梯度
- 查询目标院校招生章程中的具体级差设置
- 如果级差较大，更要慎重安排专业顺序
- 有专业级差的院校，第一专业志愿的选择尤为关键

### 分数清与专业清的对比

| 专业分配方式 | 说明 | 特点 |
|------------|------|------|
| **分数清**（分数优先） | 按分数从高到低依次安排专业 | 高分考生更有优势，级差较小或无级差 |
| **专业清**（专业优先） | 按专业志愿顺序分轮安排 | 第一专业志愿非常关键，级差影响大 |
| **专业级差** | 结合分数和专业顺序，设有级差 | 兼顾分数和志愿顺序，需权衡策略 |

## 二、服从专业调剂

### 含义

服从专业调剂指考生所填专业志愿均未被录取时，是否同意被调剂到该校其他未录满的专业。

### 选择"服从"的利弊

| 方面 | 说明 |
|------|------|
| **优点** | 大大降低退档风险；增加被该校录取的机会 |
| **缺点** | 可能被调剂到自己不感兴趣或冷门的专业；入学后转专业有一定门槛 |

### 选择"不服从"的利弊

| 方面 | 说明 |
|------|------|
| **优点** | 不会被调剂到不喜欢的专业 |
| **缺点** | 退档风险显著增加；一旦退档只能参加征集志愿或下一批次录取 |

### 建议

- **一般情况下建议服从调剂**，除非考生分数在该校有绝对优势或做好了退档/复读的心理准备
- 如果非常介意专业，可以适当降低院校层次，选择更有把握的院校保底
- 在"专业+院校"模式下（如浙江、山东、辽宁），不存在专业调剂，因为直接填报到具体专业
- 在"院校专业组"模式下（如广东、湖北），调剂通常在专业组内进行，风险相对可控

### 退档后的补救途径

1. **征集志愿**：本批次录取结束后，未完成计划的院校会公布缺额进行征集
2. **下一批次**：如本科一批退档，可参加本科二批或合并后的本科批征集
3. **复读**：如果征集志愿和下一批次均不理想，可考虑复读
