# 新高考选科与填报模式

> **免责声明**：本文档为通用知识整理，具体政策以各省教育考试院和各高校当年发布的官方招生文件为准。高考政策每年可能调整，请务必查阅最新官方信息。

---

## 一、新高考主要模式

### "3+3"模式

| 项目 | 说明 |
|------|------|
| 适用省份 | 浙江、上海、北京、天津、山东、海南 |
| 必考科目 | 语文、数学、外语（3 门） |
| 选考科目 | 从物理、化学、生物、历史、地理、政治中任选 3 门 |
| 总成绩 | 语数外各 150 分 + 选考 3 门各 100 分 = 750 分（其中上海为 660 分） |

### "3+1+2"模式

| 项目 | 说明 |
|------|------|
| 适用省份 | 河北、辽宁、江苏、福建、湖北、湖南、广东、重庆、黑龙江、甘肃、吉林、安徽、江西、贵州、广西、山西、河南、陕西、内蒙古、四川、云南、宁夏、青海 |
| 必考科目 | 语文、数学、外语（3 门） |
| 首选科目 | 物理或历史（2 选 1），按原始分计入 |
| 再选科目 | 从化学、生物、地理、政治中任选 2 门，按等级赋分计入 |
| 总成绩 | 语数外各 150 分 + 首选 100 分 + 再选 2 门各 100 分 = 750 分 |

## 二、两种填报模式的详细对比

### 院校专业组模式

**适用省份**：广东、湖北、湖南、福建、江苏、上海、北京、天津、海南

**核心特点**：

- 高校将招生专业划分为若干个"专业组"，每个组内包含若干专业
- 考生以"院校专业组"为单位填报志愿
- 同一院校的不同专业组可以有不同的选科要求
- 专业组内存在专业调剂

**填报示例**（广东物理类）：

| 志愿序号 | 院校专业组 | 包含专业 | 选科要求 |
|---------|-----------|---------|---------|
| 1 | 深圳大学（201专业组） | 计算机科学与技术、软件工程 | 物理+化学 |
| 2 | 深圳大学（202专业组） | 电子信息工程、通信工程 | 物理+化学 |
| 3 | 深圳大学（203专业组） | 土木工程、建筑学 | 物理（不限选科） |

**调剂规则**：如果档案被投到某专业组，但填写的专业均未录取，则在该专业组内进行调剂。

### 专业（类）+院校模式

**适用省份**：浙江、山东、辽宁、河北、重庆

**核心特点**：

- 直接以"专业（类）+院校"为填报单位
- 不存在专业调剂（因为已经精确到具体专业）
- 志愿数量通常较多（80-112 个）
- 每个志愿独立决定梯度定位

**填报示例**（浙江考生）：

| 志愿序号 | 专业+院校 |
|---------|----------|
| 1 | 计算机类+浙江大学 |
| 2 | 计算机科学与技术+杭州电子科技大学 |
| 3 | 软件工程+浙江工业大学 |
| ... | ... |

**调剂规则**：不设专业调剂，所有专业均为直接填报，录取即被目标专业录取。

### 两种模式的关键差异

| 对比维度 | 院校专业组模式 | 专业+院校模式 |
|---------|--------------|--------------|
| 填报单位 | 院校专业组（组内多个专业） | 专业（类）+院校（一个专业） |
| 专业调剂 | 组内调剂，风险可控 | 无调剂，无退档风险（就专业而言） |
| 志愿数量 | 相对较少（24-45 个专业组） | 数量较多（80-112 个） |
| 梯度策略 | 专业组级别 + 组内专业级别 | 每个"专业+院校"独立定位 |
| 投档风险 | 投档后退档可能 | 投档即录取（无调剂退档） |
| 选科要求 | 每个专业组独立设置 | 每个专业独立设置 |

## 三、选科要求对照

### 常见专业的选科要求

| 专业方向 | 常见选科要求（3+1+2 模式） | 说明 |
|---------|------------------------|------|
| 临床医学 | 物理+化学+生物（或物化） | 绝大多数要求物化双选 |
| 计算机/软件 | 物理+化学 | 多数要求物理，顶尖院校要求物化 |
| 电子信息 | 物理+化学 | 通常要求物理 |
| 机械/自动化 | 物理+化学 | 通常要求物理 |
| 土木/建筑 | 物理（+化学） | 多数要求物理 |
| 法学 | 不限 / 历史 | 多数不限选科 |
| 金融/经济 | 不限 / 物理 | 部分院校偏好物理 |
| 汉语言文学 | 不限 / 历史 | 多数不限选科 |
| 英语/外语 | 不限 | 通常不限选科 |
| 思想政治教育 | 政治 | 多数要求政治 |
| 地理科学 | 地理（+物/化） | 通常要求地理 |

> **注意**：以上为常见要求，各高校可能存在差异。填报前务必查阅目标院校当年的招生章程中的选科要求。

### 选科建议

- **选科已定**：根据实际选科筛选可报专业
- **选科未定**（高一/高二）：目标专业方向决定选科组合
- **冲突处理**：如果意向专业与选科不符，须主动向考生指出并建议查阅目标院校的具体要求
