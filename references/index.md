## 使用方式

在 SKILL.md 工作流的对应步骤中，已标注指向各知识文件的引用链接。AI Agent 在处理用户问题时，应：

1. 按 SKILL.md 工作流六个步骤顺序执行
2. 在执行到某一步骤时，若涉及该步骤对应的知识主题，点击引用链接查看详细内容
3. 将知识文档中的规则和说明融入对用户的回答中
4. **注意**：知识文档中的政策和数据均可能过时，AI 不应将其视为当年实时数据，应引导用户以官方文件为准

## 更新建议

- 高考政策每年可能调整，建议在每年高考季前（约 4-6 月）审阅并更新相关内容
- 如某省份推出新的志愿填报规则，应及时更新对应文档
- 新增主题文档时，同步更新本 index.md 和 SKILL.md 中的引用链接
