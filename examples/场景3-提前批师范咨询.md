# 测试场景 3：提前批 / 师范类咨询

## 场景描述

用户对师范类专业感兴趣，询问提前批的公费师范生填报事宜。AI 应介绍公费师范生的政策、提前批录取规则，并提醒一旦录取将无法参加普通批。

## 模拟用户输入

```
省份：四川
科类：文科
总分：580
位次：4500
意向专业：汉语言文学（师范）
问题：我想当老师，听说有公费师范生，和普通师范有什么区别？提前批报了会不会影响后面的普通批？
```

## 预期 AI 行为

### 第一步：信息采集与追问

> 感谢您的信息！在给您建议之前，还想了解几个信息：
> 1. 您选考的是历史类还是政治类？（四川 2025 年起实施新高考，需确认科类）
> 2. 您意向在哪个城市/地区就读？是否有地域偏好？
> 3. 您填报时更看重什么？分数优先、院校优先还是专业优先？

### 第二步：公费师范生政策解释

> **关于公费师范生与普通师范的区别：**

| 对比项 | 公费师范生 | 普通师范生 |
|--------|-----------|-----------|
| **学费** | 免学费、住宿费，补助生活费 | 正常缴纳学费 |
| **就业** | 入学即签约，毕业后到生源地中小学任教 | 自主择业 |
| **服务年限** | 通常不少于 6 年 | 无强制要求 |
| **考研政策** | 工作满一学期后可申请免试攻读非全日制教育硕士 | 正常考研 |
| **录取批次** | 提前批录取 | 普通批录取 |
| **违约后果** | 退还教育费用 + 缴纳违约金，记入诚信档案 | 无 |

> **提前批与普通批的关系：**
> - 提前批在普通批之前录取。如果您填报了提前批的公费师范生并被录取，将**不再参加后续普通批次的录取**。
> - 如果提前批未被录取，**不影响**后续普通批的正常填报和录取。
> - 因此，"不报白不报"是误区——只有在确实愿意接受公费师范生条款的前提下才建议填报。
